# PyGit

A small scale git like system implemented in Python by Khadeim Rahman. This project was designed to learn and showcase Python, SQL, and Docker skills in building a real backend system.

---

## **Project Status**

**Stage 1:** Project structure setup and `pygit init` command implemented. 
**Stage 2:** File staging system implemented with `pygit add`

## *Current Features* (In development):
- Initialise a repository with `pygit init`
- Stage files using `pygit add`
- Internal staging system using `.pygit/index.json`

## *Technologies*
- Python
- JSON
- Pytest
